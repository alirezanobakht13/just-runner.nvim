# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-05

### Added
- Initial release of just-runner.nvim
- Support for running justfile targets from Neovim
- Interactive target selection with telescope.nvim and snacks.nvim
- Configurable window positioning (bottom, right, float)
- Auto-close terminal on success/failure
- Support for targets with parameters
- Automatic justfile discovery
- Comprehensive documentation
- Test suite
- Example justfile for testing
